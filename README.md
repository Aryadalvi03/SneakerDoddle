# 👟 SneakerDoodle – E-Commerce Sneaker Website

SneakerDoodle is a modern e-commerce website designed for browsing and purchasing sneakers online. The platform provides a smooth shopping experience with product listings, search functionality, cart system, user authentication, and payment integration.

The project focuses on creating a clean and responsive UI while implementing core e-commerce functionalities.

## Features

- User signup and login system  
- Product catalog with multiple sneaker listings  
- Search functionality  
- Add to cart feature  
- Payment page integration  
- QR code payment option  
- Order and shipping pages  
- Responsive design  

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- Firebase (Authentication / Configuration)  
- Tailwind CSS  

## How to Run

Clone the repository, open the project folder in VS Code, and run the index.html file using Live Server to start the application.
